# Collector Nation — Local Setup

## Step 1: Add your Shopify token
Open `.env.local` and replace PASTE_YOUR_TOKEN_HERE with your actual token:
```
VITE_SHOPIFY_TOKEN=shpat_your_actual_token
```

## Step 2: Install Node.js (if not already installed)
Download from https://nodejs.org — pick the LTS version and install it.

## Step 3: Run the app
Open Terminal, navigate to this folder, then run:
```
npm run dev
```

## Step 4: Open in browser
Go to: http://localhost:5173

## Simulate mobile view
Chrome: Cmd + Option + I → click phone icon → iPhone 14 Pro
