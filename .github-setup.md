# One-time GitHub Setup

Run these commands ONCE in your terminal to connect your local folder to GitHub:

```bash
cd ~/Downloads/collector-nation/collector-nation

# Initialize git
git init
git add .
git commit -m "Collector Nation v1"

# Connect to GitHub (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/collector-nation.git
git branch -M main
git push -u origin main
```

After this, every time Claude makes a fix, you just run:
```bash
./update.sh
```

Or if you want to do it manually:
```bash
git pull origin main
npm install  # only needed if package.json changed
npm run dev
```
